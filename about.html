About EditKit

EditKit is a collection of browser-based tools, designed to streamline workflows in video editing. Built for editors who need reliable utilities without the overhead of complex, paid software. Features include subtitle editing tools, LUT previewing, FCPXML starters, and a project manager to come.

Why I Built This

I've been stylizing and batch producing subtitles ever since I started creating content. There weren't many free softwares that helped with my workflow, so I started writing Python scripts to handle subtitle conversions, timing adjustments, and format cleanups: mostly piecing code together with limited programming knowledge from self-teaching.

These tools come from those scripts, dating all the way back to 2022. And now that I've got more technological knowledge, I believe it was the adequate time to publish them.

Philosophy

Tools should be fast, focused, and easily accessible. Everything runs in your browser, with no installations, no sign-ups, no data sent to servers. Your files stay on your machine.

Open Source

EditKit is built with web standards and open-source technologies. Feel free to explore the code, suggest features, or contribute improvements!
